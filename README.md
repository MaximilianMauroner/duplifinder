## Getting Started

Install Node.js, Clone the repository, Install NPM packages, Start

### Prerequisites

* Install Node.js
```https://nodejs.org/en/download/ ```


### Installation

1. Clone the repo
   ```sh git clone https://github.com/MaximilianMauroner/duplifinder.git ```
2. Install NPM packages
   ```sh npm install ```
3. Start
  ``` run npm start ```
